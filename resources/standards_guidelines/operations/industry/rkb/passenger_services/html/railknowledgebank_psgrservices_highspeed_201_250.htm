Title Development of algorithms for earthquake early warning system using ocean bottom seismic network
 
Author Korenaga, M
Miyakoshi, H
Yamamoto, S
Aoi, S
Affiliation RTRI Tokyo, Japan
Source Dennis Schut (UIC)
Publication Date 2018
Abstract We have developed new algorithms to stop the Shinkansen quickly and reliably by utilizing the real-time data observed at the ocean bottom, and put them to practical use. For securing the safety of running trains at the time of earthquake, the earthquake early warning (EEW) systems that detect tremors of an earthquake and rapidly issue warning signals have been installed for Shinkansen throughout Japan. The original EEW system for railways uses data from inland seismometers. On the other hand, the world's largest ocean bottom seismometer (OBS) networks have been recently developed by governmental organizations in Japan. Since the OBSs are installed close to the epicenter of earthquakes which occur beneath the sea, it is expected that the OBSs are able to detect tremors of the earthquakes more rapidly than seismometers installed on the land. From the above background, so as to issue warning signals for earthquakes beneath the sea as soon as possible, we have developed new algorithms for EEW system using the data of the OBS networks. We have proposed a judging algorithm for EEW by using the data from OBS networks, a monitoring algorithm to prevent false alarms, and a communication algorithm to transmit data from the OBS networks to railway EEW systems. By using the actually observed data, we confirmed that the new EEW system with new algorithms can issue a warning signal about 10 sec faster than the present system. The new algorithms for OBS have been installed in some EEW system of Shinkansen line and now in operation. It is expected that EEW systems with new algorithms will increase in number to secure safer operation against earthquakes.
